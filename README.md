Git Repo
==========

git_repo is a simple git wrapper for python

REQUIREMENTS
============

* Git ( tested with 1.7.6 )
* Python Nose - used for running the tests
* Mock

SOURCE
======

https://github.com/siu/git_repo

and can be cloned using:

git clone git://github.com/siu/git_repo.git
